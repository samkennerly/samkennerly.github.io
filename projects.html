aws

ec2box
Example Terraform module for automated Amazon Web Services cloud provisioning. Each "box" is an EC2 instance with its own keypair, security group, CloudWatch log group, IAM role, and cloud-init template.

docker

blackbox
Self-contained shell script that runs Black (a Python code formatter) in a container.
dockerbash
Short shell scripts to avoid typing long Docker commands.

python

fountain
Infinite FizzBuzz solver. Includes examples of doctest, generators, and itertools.
kingofsnake
Data toolbox serving Jupyter notebooks from containers. Includes examples of cleaning, classification, clustering, graph drawing, and principal component analysis.
pydiner
Template for a Python dev environment which builds its own containers. Includes examples of automated testing, log streaming, and relative imports.

q

q_practice
Introduction to Q, the scripting language for KDB+ databases. Includes examples of dicts, dots, iterators, lambdas, projections, and flip enlist system "v".

assimilate
Automated setup chores for a new Mac: install software tools, clone repos, and create symlinks to dotfiles.
