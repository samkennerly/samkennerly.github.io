Some of my public software projects are listed here. For deep cuts, see github.com/samkennerly.

aws

ec2box
Example Terraform module for provisioning AWS EC2 instances.
Each box includes its own keypair, security group, CloudWatch log group, IAM role, and cloud-init template.

docker

dockerbash
Short shell scripts for long Docker commands.
Never type docker ps -qf ancestor="myimage" | xargs docker rm -f again.

python

blackbox
Run Black (the uncompromising Python code formatter) in a Docker container.
All code is in one shell script, including a Dockerfile heredoc.
brundling
Calculations and plots for the 7 Degrees of Martin Brundle.
An exhibition of lightweight data analysis and graph layout with pandas.
fountain
Infinite FizzBuzz generator. A harmless toy.
Includes examples of doctest, generators, and itertools.
kingofsnake
Run Jupyter in a container. Includes example notebooks.
Ensure people can replicate your studies with identical software.
pydiner
Template for a Python project which builds its own containers.
Spend less time naming things and fixing import errors.
quarto
Build websites with Python 3. No installation required.
Made by a simple script, free of state and frameworks, pip requiring nothing.

q / kdb+

q_practice
Introduction to Q, the scripting language for KDB+ databases.
Dicts, dots, iterators, lambdas, projections, and a flip enlist system "v".

ruby

jekyllbox
Run Jekyll in a Docker container.
Build static websites without installing a Ruby developer environment.
