python pdb

abstract

pdb is Python's debugger.

basics

  • pdb is a built-in Python module.
  • python -m pdb runs pdb like a script.
  • python -m pdb myscript.py runs myscript.py and starts pdb.
  • Python's breakpoint() function interrupts a script (or module) to start pdb.
