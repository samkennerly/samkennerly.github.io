python pdb

abstract

pdb is Python's built-in debugger.

basics

  • pdb is a Python module.
  • python -m pdb runs pdb like a script.
  • python -m pdb myscript.py runs myscript.py and starts pdb.
  • Python's breakpoint() function interrupts a script (or module) to start pdb.

dependencies

pdb is included with Python.

examples

Run pdb like a script:

python -m pdb

Run path/to/myscript.py and start pdb:

python -m pdb path/to/myscript.py

Start pdb from inside a Python interpreter:
