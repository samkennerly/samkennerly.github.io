q types

abstract

Data types for the q language are listed here.

basics

  • The type command returns a short int numeric code.
  • Some types also have char codes, e.g. 7h is a short int.
  • Atoms have negative codes. Lists have positive type codes.
  • Short int and char codes can be used to cast variables.

examples

Atoms have negative types. Lists have positive types.

q)type 23
-7h
q)type enlist 23
7h
q)type 1,2,3,4,5
7h
q)type (1;2;3;4;5)
7h
q)type each 1,2,3,4,5
-7 -7 -7 -7 -7h

A string is a list of chars.

q)type "Hello, World!"
10h

Cast a string as a symbol.

q)` $ "Hello, World!"
`Hello, World!
q)-11h $ "Hello, World!"
`Hello, World!

Convert a list of symbols to a list of strings.

q)type `one`two`three`four`five
11h
q)type string `one`two`three`four`five
0h
q)string `one`two`three`four`five
"one"
"two"
"three"
"four"
"five"
