RubyGems quickstart

abstract

  • Gems are self-contained Ruby packages.
  • RubyGems is a package manager for Ruby.
  • gem is the shell command which runs RubyGems.
See the RubyGems guides for details.

commands

These commmands use rake as an example gem.
install RubyGems Ruby >= 1.9 includes RubyGems.
show RubyGems environment gem env
update RubyGems sudo gem update --system
find gems by name gem search rake
find gems by regex gem search ^rake$
install a gem sudo gem install rake
install a gem version sudo gem install rake:12.3.2
uninstall a gem sudo gem uninstall rake
show installed gems gem list --local
show a gem install path gem which rake
See the RubyGems basics for details.

dependencies

Some commands may require superuser permissions.

examples

install a specific gem version

gem install nokogiri:1.5.6
RubyGems will automatically:
  • check if any gems named nokogiri are already installed
  • download nokogiri-1.5.6.gem from rubygems.org
  • uncompress the file and build any gem extensions
  • install any other gems required by nokogiri:1.5.6
  • build documentation for installed gems
See the gem install docs for details.
