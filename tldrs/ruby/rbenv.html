rbenv quickstart

abstract

  • rbenv is a shell script which manages Ruby versions.
  • ruby-build is a bash script which installs Ruby versions.
  • The ~/.rbenv/shims folder contains shim executables .
  • The ~/.rbenv/versions folder contains installed Ruby versions.
  • The ~/.rbenv/version file sets global Ruby for a user.
  • Any .ruby-version file sets local Ruby for a folder and its descendents.
  • Any RBENV_VERSION environment variable overrules global and local Ruby.
See How It Works for details.

commands

install rbenv brew install rbenv
uninstall rbenv brew uninstall rbenv
upgrade rbenv brew upgrade rbenv ruby-build
modify shell to use rbenv eval "$(rbenv init -)"
show installed Rubies rbenv versions
show available Rubies rbenv install --list
install a Ruby rbenv install 2.6.3
uninstall a Ruby rbenv uninstall 2.6.3
use an installed Ruby rbenv shell 2.6.3
revert to default Ruby rbenv shell --unset
See the Command Reference for details.

dependencies

  • Homebrew must be installed to use brew commands.
  • RVM must not be installed.
See Installation for details.

examples

Set local Ruby for a project.

cd path/to/myproject
echo '2.6.3' > .ruby-version
rbenv install --skip-existing 2.6.3
